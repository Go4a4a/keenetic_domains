Минимальный старт:

1) Установить Python 3 для Windows, если его нет.
2) Открыть PowerShell в этой папке.
3) Запустить:

python .\build.py

Готовые списки появятся в папке output:

output\telegram.txt
output\youtube.txt
output\instagram.txt
output\chatgpt.txt

Формат файлов строго для Keenetic DNS-Based Routes:
один домен на строку, без *, без DOMAIN-SUFFIX, без domain:, без full:, без комментариев.
